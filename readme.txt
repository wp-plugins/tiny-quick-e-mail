Plugin that enables you to send e-mails quickly from your wordpress.

== Description ==

Plugin that enables you to send e-mails quickly from your wordpress. You can create categories, sort your users and each admin on your blog can have its own address book with its own settings.

* Plugin first version

== Installation ==

1. Upload folder `tiny-quick-e-mail` to your `/wp-content/plugins` directory
2. Activate the plugin through the `Plugins` menu in WordPress

== Features ==

* Add contacts
* Sort them by categories
* Send e-mails from your dashboard
* Save/Edit/Delete e-mails
* Auto save open e-mail
* Add users just by clicking on them
* Full AJAX
* Custom setting for each admin

== Planned Features ==

* Send e-mail to users that have commented on your blog
* Choose who can use this plugin (for now just admins)
* E-mail revisions
* Import contacts from Gooogle, Yahoo and MSN
* Export/Import e-mails, contacts and settings
* Send files
* Add picture to contacts
